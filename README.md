## 📌 Sobre o Projeto

O Yuri-Bot-V3.5 é um bot criado para facilitar tarefas no WhatsApp, oferecendo diversos comandos como:

- 📥 Downloads (YouTube, TikTok, etc)
- 🛠️ Ferramentas úteis
- 🎮 Comandos de diversão
- 👮 Administração de grupos
- 💰 Sistema RPG (opcional)

---

## 📦 Instalação no Termux

### 1 - Atualize o Termux

```bash
pkg upgrade -y && pkg update -y
```

---

### 2 - Permitir armazenamento

```bash
termux-setup-storage
```

---

### 3 - Ir para uma pasta

```bash
cd /sdcard
```

---

### 4 - Clonar o repositório

```bash
git clone https://github.com/SEU-USUARIO/YURI-BOT3.5.git
```

---

### 5 - Entrar na pasta

```bash
cd YURI-BOT3.5
```

---

### 6 - Instalar dependências

```bash
npm install
```

---

### 7 - Iniciar o bot

```bash
node connect.js
```

---

## ⚙️ Configuração

- Edite o arquivo `config.json`
- Coloque seu número e configurações do bot
- Se usar API, adicione sua KEY
